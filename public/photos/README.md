# YoungMinds photo library

Adaugă aici pozele reale folosite de template-urile de marketing.

Nume recomandate pentru start:

- `robotica-01.jpg`
- `stem-01.jpg`
- `pian-01.jpg`
- `taekwondo-01.jpg`
- `yoga-01.jpg`
- `spatiu-joaca-01.jpg`

Reguli utile:

- format JPG sau PNG;
- ideal 4:5 sau 3:4;
- lumină bună, spațiu clar, activitate recognoscibilă;
- acorduri foto/GDPR pentru copiii recognoscibili;
- evită poze foarte aglomerate, pentru că textul peste ele devine o crimă vizuală mică, dar persistentă.

După ce adaugi pozele, verifică `lib/media/photo-library.ts` și actualizează URL-urile/temele dacă schimbi numele fișierelor.
